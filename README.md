# GhostLink 🛡️

GhostLink is a secure, 100% offline, decentralized peer-to-peer chat application designed for wartime, disaster zones, and censored environments. 
It requires **NO internet, NO phone number, NO central server, and NO registration**. Every feature works 100% offline.

## Features
- **Local Blockchain (Hash-chaining)**: Every message cryptographically links to the previous one, guaranteeing tamper-evident database logs.
- **Offline Mesh Routing**: Routes messages via Wi-Fi Direct and Bluetooth Low Energy across multiple devices.
- **Zero-Knowledge Identity**: Automatically generates Ed25519 keys locally. User IDs are completely anonymous.
- **Anti-Tracking**: Emits cover traffic, pads packet sizes, and rotates anonymous routing IDs.
- **Hardware-backed Storage**: Roots all master keys within the Android Keystore system.

## How to Test
1. Go to the **Actions** tab in this GitHub repository.
2. Click on the latest green ✅ successful build.
3. Scroll to the bottom and download the **GhostLink-App-Testing** APK file to install on your Android device.
